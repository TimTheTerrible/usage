Usage
-----

A simple utility to recursively search and list
the used space in a directory.

usage.pl is a simple perl script that recursively stats the current directory,
reporting the usaed space of each directory entry it finds.


Compiling
---------

To compile usage, just do 

    make


Installation
------------

To install usage, do

    make install

This will copy the binary and the script to the BINDIR,
as well as set the UID bit on the binary wrapper.
